# TrainerMVP

App iOS de treino personalizado com planos diários e subscrição Premium.

## O que faz
- Planos de treino personalizados por 7 dias
- Biblioteca de 50+ exercícios
- Rastreamento de progresso
- Subscrição Premium (R$9,99/mês)

## Para publicar na App Store
1. Cria conta Apple Developer: https://developer.apple.com (€99/ano)
2. Vai a App Store Connect: https://appstoreconnect.apple.com
3. Siga os passos para submeter a app
